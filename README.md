# ttygraph

A simple utility for graphing on the commandline

## Usage

     ttygraph [righthand of func]

### Examples:

     ttygraph "x"
     ttygraph "x*x"
     ttygraph "2*sin(x)"

## Compilation

     git clone "https://github.com/Cubified/ttygraph"
     cd ttygraph
     make

And optionally (installs to `~/.local/bin`):

     make install

Note:  `ttygraph` requires `ncurses`
